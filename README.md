# Love Memorandum

A small Memo for couples. 一个情侣用的简单记事本。

# 使用
在根目录下新建 users.json：
``` json
[
    ["username1", "password1"],
    ["username2", "password2"]
]
```

为什么没有注册页面？因为这是两个人用的，当然要两个人一起初始化啦。
